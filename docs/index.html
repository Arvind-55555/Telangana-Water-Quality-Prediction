Project Overview

About

Comprehensive water quality analysis using ML to classify water into Safe, Polluted, and Highly Polluted categories based on WHO/BIS standards.

Data Source

Telangana Pollution Control Board - June 2025. Includes 213 monitoring stations across rivers and lakes.

Technology

Random Forest, XGBoost, Neural Networks with 18 key water quality parameters including pH, BOD, COD, and coliform counts.

Water Quality Prediction Tool

How It Works

  1. Enter water quality parameters
  2. System calculates using WHO/BIS standards
  3. Get instant classification and WQI
  4. Receive treatment recommendations

Real-Time Calculation: This tool uses the same WHO/BIS water quality standards and weighted averaging method implemented in our ML models.

Standards Used

  • pH: 6.5-8.5 (ideal: 7.0)
  • DO: ≥5.0 mg/L (ideal: 6.0)
  • BOD: ≤3.0 mg/L
  • COD: ≤10 mg/L
  • Total Coliform: ≤50 MPN/100ml
  • TDS: ≤500 mg/L (ideal: 300)
